Язык сайта
Текущий язык сайта отображен в главном меню (сверху справа) в виде иконки соответствующий страны (английский, казахский, русский).
Для изменения языка сайта необходимо выбрать в выпадающем списке с иконкой страны желаемый язык.
Регистрация и авторизация
Регистрация новых пользователей осуществляется на странице «Регистрация», страница доступна из пункта главного меню неавторизованного пользователя «Регистрация». Для регистрации необходимо ввести электронную почту, пароль и его подтверждение.
На странице «Войти», доступной из главного меню, осуществляется авторизация зарегистрированных пользователей по электронной почте и паролю.
Дерево показателей
Для поиска данных и карт по показателям следует воспользоваться деревом показателей на главной странице сайта. При выборе блока показателей (корневой уровень дерева), группы показателей (второй уровень вложенности), показателя (третий уровень вложенности) в правой части экрана под пунктом «Тематические карты» будет отображен список ссылок с названиями карт, которые относятся к выбранному пункту дерева. Для просмотра карты достаточно перейти по ссылке с ее названием.
Для просмотра значений показателя через дерево показателей необходимо нажать на кнопку с символом «i» рядом с названием показателя в дереве.
Индексы (интегральные карты) отображаются в пункте «Индексы».
Контакты и информация о сайте
Контактная информация и информация о сайте представлены на страницах «Контакты» и «О сайте» соответственно.
Список карт
Для более точного и удобного поиска карт следует перейти на страницу «Карты», в которой присутствует список всех карт сайта. Список может быть отфильтрован по следующим полям:
•	блок показателей,
•	группа показателей,
•	показатель,
•	название (на любом языке),
•	описание,
•	ключевые слова.
После выбора, ввода ограничивающих параметров следует нажать кнопку «Найти».
Для сортировки списка карт необходимо нажать на соответствующую ссылку:
•	блок показателей,
•	группа показателей,
•	показатель,
•	название (на любом языке).
Для просмотра карты необходимо перейти по ссылке «Смотреть» напротив соответствующей карты в списке.
Просмотр карты
На странице просмотра карты размещается:
•	название карты (в верхней части),
•	меню карты (в левой части),
•	интерактивная карта (в правой части).
Меню карты состоит из:
•	легенды к карте в виде цветового индикатора и текстового описания пункта;
•	легенда к пунктам диаграмма (если таковая имеется на карте);
•	выпадающий список типов диаграммы (круговая, круговая трехмерная, кольцевая, гистограмма);
•	выпадающего списка типов базовых слоев (OpenStreetMap, ArcGIS, Bing, BingAerial, Stamen);
•	полосы прокрутки изменения прозрачности базового слоя и слоя данных (от 0% до 100%);
•	кнопка поиск (вызывает меню поиска областей, значения которых за выбранный год будут соответствовать введенным данным);
•	полоса прокрутки года (при изменении происходим перерисовка карты со значениями выбранного года).
Для получения информации об области необходимо кликнуть по ней на карте. В появившемся окне будет представлена следующая информация:
•	название области,
•	значения основного показателя (за выбранный год),
•	значения показателей диаграммы (если таковая имеется на карте),
•	разницы (отклонения) показателей от среднереспубликанских значений.
При клике на значение основного показателя или его отклонения от среднереспубликанского показателя вызывается окно с просмотром диаграммы изменения значения основного показателя и его отклонения от среднереспубликанского значения по выбранной области за все годы.
При клике на значение показателя диаграммы или его отклонения от среднереспубликанского показателя вызывается окно с просмотром диаграммы изменения значения показателя диаграммы и его отклонения от среднереспубликанского значения по выбранной области за все годы.
Отчеты
На данной странице представлены ссылки на три вида отчетов:
•	по региону,
•	по показателям,
•	по годам.
Отчет по региону дает возможность получить отчет по области, по трем показателям и трем годам. Отчет по показателям дает возможность получить отчет по показателю, трем областям и трем годам. Отчет по годам дает возможность получить полный отчет по показателю за все годы и по всем областям.
Для формирования отчета необходимо выбрать соответствующие параметры и нажать кнопку «Скачать».
Все отчеты формируются в формате MS Excel и доступны для скачивания.
